## Como executar

Pré-requisitos: 
- Visual Studio Code
- GIT

```bash
# clonar repositório
git clone https://github.com/maxxdiego/11_typescript.git

```

```bash
# Instalar todas as dependências indicadas pelo package.json
npm install

```

```bash
# Rodar o projeto React
npm run dev

```

```bash
# No navegador acessar a aplicação através do 
http://localhost:3000

```
